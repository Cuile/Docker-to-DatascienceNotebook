# cn-datascience-notebook

支持中文显示的 Jupyter Notebook

## 镜像说明

本镜像基于 jupyter/datascience-notebook 制作，添加中文显示的能力。

解决了使用绘图能力时，中文显示乱码问题。

## 版权声明

本镜像为方便个人学习研究数据分析为目的。

本镜像使用 “微软雅黑 YaHei Consolas Hybrid 1.12” 字体，解决中文显示问题，但微软字体禁止在其它系统上使用，请查看具体的协议内容。
